**OmniZ Server** — это легкое, портативное и современное приложение для управления серверами DayZ. Создано для администраторов, которые ценят минимализм, удобство и надежность.

## ✨ Возможности (v0.1)

* 🟢 **Управление в один клик:** Быстрый старт, остановка и перезагрузка сервера.
* ⏱️ **Авторестарт:** Встроенный таймер для автоматической перезагрузки сервера (каждые 3, 6 или 12 часов).
* ⚙️ **Гибкая конфигурация:** Удобная боковая панель с предзаполненными стандартными параметрами запуска (port, config, profiles, cpuCount и др.).
* 🌐 **Локализация:** Поддержка русского (🇷🇺) и английского (🇺🇸) языков "на лету".
* 🛡️ **Безопасный UI:** Интерфейс адаптирован и защищен от искажений при использовании темных тем Windows.
* 🔄 **Умные обновления:** Фоновая проверка новых версий с уведомлением администратора.

## 🚀 Как запустить
1. **Скачайте программу:**
   Перейдите в раздел [Releases](../../releases) справа и скачайте последнюю версию `OmniZ_Server.exe`.
2. **Запустите файл:**
   Положите `.exe` файл в любую удобную папку (например, на Рабочий стол) и запустите его двойным кликом.
3. **Укажите путь к серверу:**
   При первом запуске нажмите на **шестеренку** ⚙️ в правом верхнем углу. Нажмите «Обзор» и выберите файл вашего сервера (обычно это `DayZServer_x64.exe`).
4. **Старт!**
   Нажмите большую зеленую кнопку **«СТАРТ»**. Программа автоматически применит правильные параметры запуска, а сервер появится в сети.

*⚠️ Важно: Так как программа новая и не имеет цифровой подписи от Microsoft, антивирус Windows (SmartScreen) может выдать предупреждение при первом запуске. Просто нажмите «Подробнее» -> «Выполнить в любом случае».*
